# Simple E-Commerce Admin Panel

This is a simple command-line admin panel written in Python. It allows basic operations:
- Add products
- List products
- Delete products by ID

## Technologies Used
- Python 3.x

## How to Run
```bash
python3 ecommerce_admin_panel_initial.py
```
